# Spreadsheet Deduper

Dedupe files via a web interface

### Setup

*Install OS level dependencies:* 

* Python 2.7
* Redis

*``pip install`` app requirements*

```bash
$ pip install "numpy>=1.6"
$ pip install -r requirements.txt
```
