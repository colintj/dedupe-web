About

What is this?

Spreadsheet Deduper is an online tool for finding and removing duplicates from a spreadsheet. It uses the dedupe entity resolution library to quickly and accurately identify duplicate rows based on training data that you give it.

Code

All of the code for this project is open source. You can find, view and fork it on github.

Technologies used:

  • Dedupe - open source entity resolution library
  • Flask - web framework
  • Bootstrap - HTML/CSS layout
