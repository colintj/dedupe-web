What is this?

Spreadsheet Deduper is a tool for identifying and removing duplicates from a spreadsheet. It uses the dedupe library to quickly and accurately identify duplicate rows based on training data that you give it.

Limitations

De-duplication takes a lot of computing power, which increases as your data gets bigger. To keep things manageable on the web, we limit the size of your file to 5mb.

If you have a large, complicated or sensitive dataset you need de-duplicated, you can:

  • download and install the dedupe library
  • hire DataMade to set up and run a custom deployment of dedupe

Code

All of the code for this project is open source. You can find, view and fork it on github.

Technologies used:

  • Dedupe - open source entity resolution library
  • Flask - web framework
  • Bootstrap - HTML/CSS layout
